# DDL (Dialogue Design Language) Archives

## Overview
Collection of experimental protocols for structural control and cognitive framing of Large Language Models (LLMs).
These files are designed to override default agentic behaviors and enforce specific reasoning modes.

## Contents structure
* **Root (PDFs)**: Observation logs and theoretical analysis.
* **Context_Framing/**: Executable definition templates for Gemini/GPT (Markdown).
